# Agent Chats

Use this directory for durable delivery evidence when a change is meaningful
enough that another agent or reviewer should be able to reconstruct it.

Recommended path:

`agent_chats/YYYY/MM/DD/YYYY-MM-DDTHH-MM-SSZ-topic.md`

Each entry should include:

- Prompt
- Scope
- Changes
- Validation
- Runtime Evidence
- Risks
- Follow-ups
- Linked commits or PRs when available
