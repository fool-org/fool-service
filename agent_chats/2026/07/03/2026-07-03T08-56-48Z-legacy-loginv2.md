# Legacy loginv2

## Prompt

Continue the Docker/FoolFrame/Vue migration with timely atomic commits.

## Scope

- Added `/api/v1/auth/loginv2` with the legacy request fields used by the old
  Cloud-Social web client.
- Reused the migrated check-code service, app catalog, app/database relation,
  and existing admin login/token flow.
- Returned legacy-style login payload fields: `loginSucess`, `token`, `user`,
  `app`, and `error`.
- Added the Vue operator-console `Legacy Login V2` action.
